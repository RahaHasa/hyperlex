## 🔧 Как добавить новые адаптивные стили

Используйте эти breakpoints:

```css
/* Планшеты */
@media (max-width: 768px) {
    /* ... стили ... */
}

/* Обычные телефоны */
@media (max-width: 640px) {
    /* ... стили ... */
}

/* Маленькие телефоны */
@media (max-width: 480px) {
    /* ... стили ... */
}

/* Очень маленькие экраны */
@media (max-width: 360px) {
    /* ... стили ... */
}
```

## ⚡ Советы для мобильного дизайна

1. **Touch-friendly размеры**
   - Минимум 44x44px для кнопок
   - Font-size >= 16px для inputs

2. **Читаемость**
   - Line-height >= 1.5
   - Достаточное расстояние между элементами

3. **Производительность**
   - Используйте `overflow-x: auto` вместо горизонтального скролла
   - Минимизируйте большие изображения
   - Используйте CSS вместо JS для анимаций

4. **Viewport**
   - Уже установлен в `index.html`:
   ```html
   <meta name="viewport" content="width=device-width, initial-scale=1.0">
   ```

## 📞 Часто встречающиеся проблемы

### Проблема: iOS зумит при клике на input
**Решение:** Font-size >= 16px на inputs ✓ (уже добавлено)

### Проблема: Текст не умещается
**Решение:** Используйте `word-break: break-word` и `overflow-wrap: break-word`

### Проблема: Кнопки слишком маленькие
**Решение:** Min-height 40-44px для всех интерактивных элементов ✓ (уже добавлено)

### Проблема: Горизонтальный скролл
**Решение:** Проверьте `max-width` контейнеров и используйте `overflow-x: auto` для таблиц
